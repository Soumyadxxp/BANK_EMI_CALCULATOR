🏦 ICICI Bank EMI Amortization Planner
A modern, professional, and fully responsive EMI Calculator & Loan Amortization Planner built using HTML, Tailwind CSS, JavaScript, Chart.js, jsPDF, and SheetJS.
This application helps users calculate their loan EMI, analyze repayment schedules, visualize loan balances, and export reports in PDF or Excel format.
